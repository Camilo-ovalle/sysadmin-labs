## Dos fases, ocho laboratorios

<CardGrid stagger>
  <Card title="Fase 1 — Active Directory" icon="laptop">
    Labs 01–04: Monta un dominio `corp.local` desde cero, configura GPOs de
    seguridad, implementa RBAC con AGDLP y automatiza la gestión de usuarios.
  </Card>
  <Card title="Fase 2 — SCCM / MECM" icon="setting">
    Labs 05–08: Instala y configura Microsoft Endpoint Configuration Manager,
    despliega software, gestiona parches con ADR y automatiza imágenes OS con
    OSD + PXE.
  </Card>
  <Card title="Infraestructura de lab" icon="seti:config">
    Todo corre sobre VMs locales en una red `192.168.100.0/24`. Fase 1 necesita
    solo 6–8 GB RAM; Fase 2 requiere 16 GB mínimo.
  </Card>
  <Card title="Comandos de diagnóstico" icon="magnifier">
    Cada lab incluye sección de troubleshooting y los comandos clave:
    `gpresult`, `dcdiag`, `repadmin` y ciclos de cliente SCCM.
  </Card>
</CardGrid>
